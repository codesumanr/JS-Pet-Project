Basic of HTML

1. Headings

Headings are used to define the importance of the text and are denoted by the <h1> to <h6> tags. <h1> is the largest (most important), and <h6> is the smallest (least important).

        
          <h1>This is a Heading 1</h1>
          <h2>This is a Heading 2</h2>
          <h3>This is a Heading 3</h3>

2. Paragraphs

The <p> tag is used for paragraphs

This is a paragraph of text.

3. Links

Links are created using the <a> (anchor) tag. The href attribute specifies the URL of the link.

        <a href="example.com">Click here</a>

4. Images

<img> tags are used to display images.

        <img src="image.jpg" alt="A beautiful image">
